# Busca linear

### Introdução

A busca linear, também conhecida como busca sequencial, é um algoritmo usado para encontrar um valor dentro de uma lista ou um array. Este método de busca verifica sequencialmente cada elemento da lista até que uma correspondência seja encontrada ou até que todos os elementos tenham sido examinados.

Para compreender a busca linear em TypeScript, vamos considerar um exemplo básico. Suponha que tenhamos um array de números:

```typescript
const numeros: number[] = [10, 5, 8, 2, 6, 3]
```

Agora, digamos que queremos encontrar o índice de um número específico neste array, por exemplo o 6. Escreveríamos uma função de busca linear assim:

```typescript
function buscaLinear (array: number[], alvo: number): number {
  if (array.length === 0) return -1 // Retorna -1 se o array estiver vazio

  for (let i = 0; i < array.length; i++) { // Percorre o array
    if (array[i] === alvo) { // Verifica se o elemento atual é igual ao alvo
      return i // Retorna o índice se o alvo for encontrado
    }
  }

  return -1 // Retorna -1 se o alvo não for encontrado
}
```

Vamos usar esta função para encontrar o índice do número 6 no array de números:

```typescript
const indiceAlvo = buscaLinear(numeros, 6)
console.log("Índice do número 6:", indiceAlvo) // Saída: Índice do número 6: 4
```

A função retorna 4, que é o índice do número 6 no array de números.

A busca linear é como quando você olha cada carta em um baralho até encontrar a carta que está procurando. Porém, se o
baralho for muito grande, pode demorar bastante. Quanto mais cartas tiver, mais tempo leva para encontrar a carta certa.
Outros métodos de busca podem ser mais rápidos, mas a busca linear ainda é importante para entender como os outros
funcionam.

### Outros métodos para a busca linear

#### Usando o método `findIndex`:

O método [findIndex](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/findIndex)
procura por um elemento no array que satisfaça a função de teste fornecida. Veja como você pode usar o findIndex:

```typescript
function buscaLinearFindIndex (arr: number[], alvo: number): number {
  return arr.findIndex(num => num === alvo)
}
```

É possível usar outros métodos como [find](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find)
ou [for...of](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/for...of), mas o findIndex é
o mais eficiente para este caso porque ele para quando encontra a primeira ocorrência e porque já busca pelo índice,
que é geralmente o resultado esperado por um algoritmo de busca linear.

### Casos de uso

A busca linear é comumente usada em cenários onde você precisa encontrar um elemento específico dentro de um array ou uma coleção de elementos. Aqui estão alguns casos de uso típicos:

1. **Arrays pequenos**: A busca linear é eficaz para arrays pequenos onde o custo adicional de implementar algoritmos de busca mais complexos, como a busca binária, seria maior do que os benefícios.
2. **Arrays não ordenados**: Quando o array não está ordenado, a busca linear continua sendo uma escolha direta. Ordenar o array antes de realizar uma busca poderia introduzir complexidade e gastos de performance desnecessários, tornando a busca linear uma opção mais prática.
